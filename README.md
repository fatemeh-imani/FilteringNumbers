# FilteringNumbers - Phase 3
.هدف 

هدف اصلی این مرحله افزایش انعطاف‌پذیری و کاهش وابستگی


## توضیحات پروژه

این پروژه یک برنامه ساده برای خواندن، فیلتر کردن و نمایش اعداد است.  
در **مرحله سوم** پروژه، متد فیلتر اعداد به گونه‌ای بازنویسی شده که بتوان **شرط فیلتر دلخواه** را به آن داد، بدون تغییر متد اصلی. 
وابستگی به شرط خاص کاهش یافته و کد قابل توسعه، تست و نگهداری شده است.
از **Strategy Pattern** استفاده شده تا انعطاف‌پذیری افزایش یابد.

---

## امکانات اضافه شده در مرحله 3

- متد `FilterNumbers` اکنون **یک اینترفیس `IFilter` می‌پذیرد**.
- می‌توان هر شرط فیلتر دلخواه را با ایجاد یک کلاس که `IFilter` را پیاده‌سازی می‌کند، تعریف کرد.
- مثال‌ها:
  - اعداد بزرگ‌تر از 10
  - اعداد زوج و بزرگ‌تر از 10
  - هر شرط دلخواه دیگر
